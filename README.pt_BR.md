# L10n Portfolio

Repositório com o histórico das minhas contribuições de localização (L10n) e tradução para projetos de código aberto.

Criei este espaço para documentar as decisões terminológicas que tomo durante as traduções. Meu objetivo é que ele sirva de referência para meus futuros Pull Requests e auxilie outras pessoas que desejam colaborar com a localização desses mesmos projetos.

---

## Sobre

Sempre fui entusiasta do ecossistema open-source e acredito que uma **localização (L10n)** bem feita é uma das formas mais diretas de tornar a tecnologia acessível. No momento, tenho focado em ajudar com a adaptação de interfaces, especialmente no ecossistema GNOME/Adwaita.

Acredito que minha formação me dê uma perspectiva melhor de escopo ao traduzir o software. Como estudo inglês desde muito pequeno, também já sou bastante familiarizado com as estruturas e termos. Mesmo assim, quando não consigo pensar em uma tradução adequada, não sou contra pesquisar em dicionários da língua inglesa/portuguesa ou então na internet por ferramentas web.

Diferente de uma tradução literal, entendo a localização como o processo de adaptar o software para que ele pareça nativo ao contexto brasileiro. Por isso, busco manter um padrão profissional e técnico, evitando termos informais ou traduções literais que não condizem com uma interface de usuário de alta qualidade. Cada pasta neste repositório contém os registros das mensagens de Pull Request que enviei, detalhando as escolhas terminológicas e as padronizações aplicadas.
