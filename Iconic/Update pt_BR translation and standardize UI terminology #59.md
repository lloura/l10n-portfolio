## [Update pt_BR translation and standardize UI terminology](https://github.com/youpie/Iconic/pull/59)

This commit updates the Brazilian Portuguese (pt_BR) translation to achieve 100% coverage, including all new strings from the latest version of Iconic.

The primary goal was to ensure a clear, native, and professional interface for Brazilian users, maintaining consistency with the GNOME HIG (Human Interface Guidelines) and the application's specific features.

---

### Key Standardization Summary

This section highlights the most significant terminology adjustments applied to ensure interface consistency:

1. **Core Feature Terms:**
    - `Regenerate` has been standardized as **"Regenerar"** (keeping the technical meaning of the folder color update process).
    - `Accent Color` translated as **"Cor de destaque"** (standard GNOME terminology).
    - `Layer (Top/Bottom)` standardized as **"Camada (Superior/Inferior)"**.


2. **UI Verbs & Actions:**
    - `Reset` standardized to **"Redefinir"** (replacing the more literal "resetar").
    - `Discard` translated as **"Descartar"** (standard for unsaved changes dialogs).


3. **Contextual Adaptations:**
    - **"Built-in"** images translated as **"Integradas"**.
    - **"Threshold"** (for monochrome) translated as **"Limite"**.
    - Maintained **"Iconic"** and **"Adwaita"** as proper nouns.


4. **Maintenance & Improvements:**

    - Removed all unused 'fuzzy' and commented lines (`#~`) to improve maintainability and readability. 
  Total line count reduced from **1214** to **1134**.
